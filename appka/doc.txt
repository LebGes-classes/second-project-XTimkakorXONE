Документация проекта "Система управления складом и продажами"
Общее описание
приложение представляет собой консольную систему управления складом и продажами, разработанную на Java. Система позволяет управлять товарами, заказами, клиентами, складами, пунктами продаж и сотрудниками.
Структура проекта
com.my_app.app - точка запуска приложения
com.my_app.entities - сущности 
com.my_app.service - сервисный слой
com.my_app.storage_service - сервисы хранения данных
com.my_app.utils - вспомогательные классы

Основные модули
1. Управление товарами
Добавление новых товаров
Просмотр списка товаров
Поиск товаров по ID
Обновление информации о товарах
Удаление товаров
Управление количеством товаров
2. Управление заказами
Создание новых заказов
Просмотр всех заказов
Поиск заказов по ID
Добавление/удаление товаров в заказе
Изменение статуса заказа
Расчет общей стоимости заказа
Удаление заказов
3. Управление клиентами
Добавление новых клиентов
Просмотр списка клиентов
Поиск клиентов по ID
Обновление информации о клиентах
Удаление клиентов
Просмотр заказов клиента
Управление балансом клиента
Просмотр транзакций клиента
4. Управление складами
Добавление новых складов
Просмотр списка складов
Поиск складов по ID
Обновление информации о складах
Удаление складов
Управление ячейками склада
5. Управление пунктами продаж
Добавление новых пунктов продаж
Просмотр списка пунктов продаж
Поиск пунктов продаж по ID
Обновление информации о пунктах продаж
Удаление пунктов продаж
6. Управление сотрудниками
Добавление новых сотрудников
Просмотр списка сотрудников
Поиск сотрудников по ID
Обновление информации о сотрудниках
Удаление сотрудников
Изменение места работы сотрудника
Консольный интерфейс с интерактивным меню
Поддержка транзакций для клиентов
